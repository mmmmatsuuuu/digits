## 背景・目的

授業内でタイピングゲーム「寿司打」に生徒が自発的に熱中する様子から着想。  
同様のゲームメカニクスをデジタル計算トレーニングに応用することで、  
「やれ」と言わなくても生徒が自主的に反復練習に取り組む状態を目指す。

## 意思決定ログ

| # | 質問 | 決定 | 理由・補足 |
|---|------|------|-----------|
| 1 | 対象デバイス | **PC（Chromebook）** | 授業での利用がChromebook中心 |
| 2 | ランキング機能 | **不要（今は）** | ミニマルに始める。サーバー実装を避けローカルで完結 |
| 3 | 答え方 | **タイピング入力** | 計算力とキーボード操作を同時に鍛える |
| 4 | プロトタイプ対象 | **情報量計算のみ** | 答えが短い数字で実装しやすく、ゲームとの相性が高い |
| 5 | ゲームコンセプト | **サイバー防衛に絞る** | 3種プロトタイプ比較の結果、最も生徒に刺さりそうと判断 |
| 6 | テーマ | **ハッカーがサーバーへ侵入を試みる** | 情報セキュリティとの文脈的一致（ビット数 = パスワード強度等） |
| 7 | 問題表示方式 | **B方式（問題1つ、最前列ハッカーを撃退）** | プロトタイプとして最もシンプル |
| 8 | 不正解ペナルティ | **★2/★3のみ: ターゲットハッカーが一時加速** | ★1はペナルティなし。加速で間接的プレッシャーを与える |
| 9 | HP設定 | **★1: 5層、★2/★3: 3層** | 問題難易度で調整し、HP差は最小限に |
| 10 | スコア計算 | **撃退時x座標 × 難易度倍率** | 右手で倒すほど高得点。難易度 ★1×1 ★2×2 ★3×4 |
| 11 | ゲームモード選択画面 | **サイバー防衛のみ（タイマーサバイバル・ビットタワーは非表示）** | 1本に集中して磨く |

## ゲームフロー（確定版）

```
TopPage（ゲームモードバナー）
  ↓ #/game
GameSelect（サイバー防衛カード）
  ↓ #/game/hacker
カテゴリ選択
  ├─ 情報量計算（プレイ可能）
  └─ 進数変換（準備中）
    ↓ カテゴリ選択
ステージ選択（★1 / ★2 / ★3）
  ↓ スペースキー or スタートボタン
カウントダウン（3 → 2 → 1 → GO!）
  ↓
ゲーム本編
  ↓ サーバー陥落（HP=0）
ゲームオーバー（リザルト + リトライ）
```

## ゲーム仕様（確定版）

### ステージ設定

| | ★1 EASY | ★2 NORMAL | ★3 HARD |
|---|---|---|---|
| ファイアウォール | 5層 | 3層 | 3層 |
| 問題難易度 | 難易度1 | 難易度2 | 難易度3 |
| 不正解ペナルティ | なし | ターゲット加速（×1.6） | ターゲット激加速（×2.2） |
| スコア倍率 | ×1 | ×2 | ×4 |
| 初期スポーン間隔 | 4.0秒 | 3.5秒 | 3.0秒 |
| 最小スポーン間隔 | 1.5秒 | 1.2秒 | 1.0秒 |

### スコア計算

```
撃退時スコア = Math.floor(hacker.x) × 難易度倍率
```
- x = 画面右端を97として、右寄りで倒すほど高得点
- 上限なし。難易度が上がるほどスコアが高くなる設計

### 難易度上昇ルール（全ステージ共通）

- ハッカー速度: 20秒ごとに +2%/秒（上限20%/秒）
- スポーン間隔: 10秒ごとに -400ms（各ステージの最小値まで）

## UIレイアウト（確定版）

```
[ヘッダー: ←戻る | ★ステージ WAVE X | SCORE: N | ■■■ファイアウォール残量]
[ゲームエリア（画面の大半）]
  [左80px: 🖥️SERVER + ■ファイアウォールブロック]
  [右側フィールド: 🧑‍💻ハッカーが右から左へ移動]
    最前列に 🎯 TARGET 表示
    正解時に +Xpt のポップアップが浮かび上がる
[問題エリア（下部固定）]
  [問題文] [単位]
  [入力フィールド] [撃退ボタン]
  [フィードバックテキスト: BLOCKED! / 不正解 / SPEED UP!]
```

### 演出一覧

| タイミング | 演出 |
|-----------|------|
| 正解 | 緑フラッシュ + `+Xpt` ポップアップ（倒した位置に浮かぶ） + `✅ BLOCKED!` |
| 不正解（★1） | 赤フラッシュ + `❌ 不正解 答え: X` |
| 不正解（★2/★3） | 赤フラッシュ + `❌ 不正解 → SPEED UP!` + ハッカーアイコンが 🏃 に変化 + 赤いグロー |
| ファイアウォール被弾 | 画面シェイク + `🔓 FIREWALL HIT!` ポップアップ + ブロック1個消失 |
| ゲームオーバー | モーダル表示（最終スコア + リトライ / ステージ選択へ） |
